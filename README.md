Descripción del Trabajo

El proyecto Calculadora_TI es una aplicación de consola desarrollada en C# que simula una calculadora básica. Permite al usuario realizar operaciones matemáticas fundamentales como suma, resta, multiplicación y división. Este proyecto es ideal para comprender la lógica de programación y la interacción con el usuario a través de la consola.

Preguntas Prácticas

1. ¿Qué operaciones matemáticas permite realizar la calculadora?

Suma, resta, multiplicación y división.

2. ¿Cómo interactúa el usuario con la aplicación?

A través de la consola, ingresando los números y seleccionando la operación deseada.

3. ¿Qué sucede si el usuario intenta dividir por cero?

Idealmente, la aplicación debería manejar este caso y mostrar un mensaje de error, aunque depende de la implementación específica.

4. ¿Es posible extender la calculadora para incluir más operaciones?

Sí, se pueden agregar funciones como potencia, raíz cuadrada, etc., modificando el código fuente.

5. ¿Qué estructura de archivos tiene el proyecto?

Incluye archivos como .gitattributes, .gitignore y Calculadora_TI.sln, indicando una solución de Visual Studio.
